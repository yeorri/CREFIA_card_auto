# 여신금융협회 카드자료 자동 다운로드

여신금융협회 가맹점 매출거래정보 통합조회(`cardsales.or.kr`)에서 여러 거래처(계정)의
**카드 매입내역 엑셀을 자동으로 일괄 다운로드**하고, 카드사별로 정리하는 도구.

## 구성

이 사이트는 자동화(CDP/Selenium 등)를 탐지해 차단하므로 두 부분으로 나뉜다:

- **Python GUI (`gui.py`)** — 거래처 목록·기간 관리, 로그인 타이핑(OS 스캔코드 입력),
  계정 순회, 다운로드 파일 정리. 로그인 입력은 OS 키 입력이라야 사이트 봇검사를 통과한다.
- **Chrome 확장 (`extension/`)** — 매입내역 페이지에서 조회→체크→상세조회→엑셀 다운로드 수행.
  페이지 내부 스크립트라 탐지되지 않으며 DOM을 정밀 제어한다.

## 설치 (PC당 1회 — 관리자)

1. **Chrome 확장 설치**: `chrome://extensions` → 개발자 모드 ON →
   "압축해제된 확장 프로그램을 로드" → `extension` 폴더 선택.
2. **실행 파일**: `dist\여신금융협회_카드자료_자동다운로드.exe` 실행 (또는 `python gui.py`).

## 사용법

1. Chrome 창 1개를 열어둔다. **영문 입력 상태**로.
2. 프로그램에서 거래처(업체명/아이디/비번)를 **직접 추가**하거나 **엑셀/CSV 불러오기**
   (A열 업체명, B열 아이디, C열 비번).
3. 조회 방식 선택:
   - **매입일자 기간조회**: 시작일~종료일(YYYYMMDD, 최대 31일).
   - **월 조회(거래일 기준)**: 연/월 선택 → 2구간 자동 조회 후 거래일(승인일) 기준으로
     해당 월만 남겨 카드사별 정리.
4. 저장 폴더 지정 후 **자동 다운로드 시작**. 실행 중에는 마우스/키보드를 건드리지 말 것.

결과 파일: `[정리]업체명_기간.xlsx`(카드사별 정렬+합계 수식) / `[원본]업체명_기간.xls`.

## 개발 / 빌드

```
pip install -r requirements.txt
python gui.py            # 실행
build.bat                # exe 빌드 (dist\)
```

- `orchestrate.py` — 계정 순회·로그인·다운로드 오케스트레이션
- `tools/os_type_scancode.py` — OS 스캔코드 키 입력(로그인용)
- `postprocess.py` — 엑셀 후공정(카드사별 정렬·합계, 월조회 합치기+거래일 필터)
- 계정 파일 형식은 `accounts.example.csv` 참고. **실제 `accounts.csv`는 커밋 금지(.gitignore).**
